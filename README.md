# HR Attrition Analysis
## Tools Used: SQL, Excel

## Project Summary
Analyzed 1,470 employee records to identify 
key attrition drivers using SQL and Excel.

## Key Findings
- Overall attrition rate: 16%
- Sales dept highest attrition: 21%
- Low salary employees leave 3x more (29%)
- Employees under 30 leave at 28% vs 11% 
  for above 40 age group
